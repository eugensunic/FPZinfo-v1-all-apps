1. Napišite program koji izračunava drugu katetu ukoliko su hipotenuza i
prva kateta poznate(koristiti klasu Math.)

Riješenje:

static void Main(string[] args)
{
int a = 4; // pretpostavka jel si napiso da je poznata jedna kateta

Console.ReadLine(); // da se program ne zatvori do pritiska enter tipke
}

3. Napisati program gdje korisnik preko tipkovnice unosi 3 broja te na temelju njih izracunajte
poluopseg te površinu trokuta preko heronove formule

static void Main(string[] args)
{
Console.WriteLine("Unesite prvi broj tj. stranicu a:"); // reda radi
int a = Convert.ToInt32(Console.ReadLine());
Console.WriteLine("Unesite drugi broj tj. stranicu b:");
int b = Convert.ToInt32(Console.ReadLine());
Console.WriteLine("Unesite treci broj tj. stranicu c:");
int c = Convert.ToInt32(Console.ReadLine());
double s = (a + b + c) / 2; // double jer postoji mogucnost dobivanaja decimalnog zbog djeljenja!
double S= Math.Sqrt(s*(s-a)*(s-b)*(s-c)); // obavezno double jel math klasa ne prihavaca int
Console.WriteLine("Povrsina preko heronove formule je "+S);
Console.ReadLine();
}

5.Napisite program koji ucitava 2 cjela broja te zatim definira granice.
(Primjer: učitani brojevi 3,7,,, donja granica je 3, gornja granica je 7)
Zatim program ispisuje brojeve koji su djeljivi sa 3 ili sa 5 u tom rasponu
Ukoliko je prvi uneseni broj veci od drugog, preokrenuti poredak da prvi uvijek
bude manji a drugi uneseni veci(zbog granica)

static void Main(string[] args)
{
Console.WriteLine("Upisite prvi broj za interval");
int a = int.Parse(Console.ReadLine());
Console.WriteLine("Upisite kranjni broj za interval");
int b = int.Parse(Console.ReadLine());
Console.WriteLine("Interval je [" + a + "," + b + "]");
if (a > b)
{
int x = b;
b = a;
a = x;
}
while (a < b)
{
if ((a % 3 == 0))
{ Console.WriteLine("broj je djeljiv sa 3 " + a);
}
else if ((a % 5 == 0))
{
Console.WriteLine("broj je djeljiv sa 5 " + a);
}
a++;
}
Console.ReadLine();
}

6. Napišite program koji generira 22 slucajna broja u rasponu od 10-25
te nakon toga ispisuje tih 22 broja ali prije brise broj na indeksu(adresi)
broj 12.

static void Main(string[] args)
{
Random rnd = new Random(); // instanca klase random , kreiranje klase
List Lista = new List(); // sintaksa za listu
for (int i = 0; i < 22; i++) // punjenje liste sa 22 elemenata
{
Lista.Add(rnd.Next(10,25)); // dodavanje 22 random brojeva u listu sa naredbom . Next(raspon)
}
foreach (int a in Lista)
{
Console.WriteLine(a);
} // ispis tih 22 broja
Console.WriteLine("----------------------------"); // cist radi preglednosti
Lista.RemoveAt(12); // brisanje elementa na indexu 12 odnosno brisanje 13-tog elementa u listi.
foreach (int a in Lista)
{
Console.WriteLine(a);
} // ispis tih 22 broja bez 13-tog odnosno bez elemnta indexa 12.
Console.ReadLine();
}
